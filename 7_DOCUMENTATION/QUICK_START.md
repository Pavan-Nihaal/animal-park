# ⚡ QUICK START (5 min)

## 1. Setup (2 min)
```bash
chmod +x 1_SETUP_SCRIPTS/setup.sh
./1_SETUP_SCRIPTS/setup.sh
```

## 2. Start local services
```bash
docker compose up -d
```

## 3. Run in dev mode
```bash
npm run dev
```

## 4. Access
- Mongo Express: http://localhost:8081
- API: http://localhost:8000
